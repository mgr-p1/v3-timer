# v3-timer
[こちらからどうぞ](https://mgr-p1.github.io/v3-timer/timer.html)  
v3を3回実行するとロックかかるので作りました。

## 操作方法

### iPhone Android の場合
v3したらどこでもいいので画面タップ

### PCの場合
v3したらどこでもいいので画面クリック

## 仕様
画面2回タップした状態で30秒経過していないと画面が赤くなります。  
画面が赤い状態でv3を発動させるとロックされると思います。

タップ時、画面下の方が1秒ほど色が変わります。
色が変わっている間はタップしてもカウントはされません。
(ミスって連続タップしてしまう対策)
